# gdate 
## chatgpt 
gdate是GNU操作系统中的日期命令，用于显示或设置系统日期和时间。

该命令的语法如下：

```
gdate [选项] [格式]
```

选项可以是以下几种：

- `-s`：设置系统时间。后面必须跟日期时间参数。
- `-u`：显示协调世界时（UTC/GMT）。
- `-R`：显示RFC 2822格式的日期时间。
- `-I`：显示ISO 8601格式的日期时间。

格式指定日期和时间的显示格式，可以使用以下格式化字符：

- `%a`：星期几的缩写
- `%A`：星期几的全称
- `%b`：月份的缩写
- `%B`：月份的全称
- `%c`：本地日期时间
- `%C`：世纪
- `%d`：日（01-31）
- `%D`：日期（MM/DD/YY）
- `%e`：日（1-31），以空格补齐
- `%F`：日期（YYYY-MM-DD）
- `%g`：ISO 8601格式的年份
- `%G`：ISO 8601格式的年份
- `%h`：月份的缩写
- `%H`：小时（00-23）
- `%I`：小时（01-12）
- `%j`：一年中的第几天
- `%k`：小时（0-23），以空格补齐
- `%l`：小时（1-12），以空格补齐
- `%m`：月份（01-12）
- `%M`：分钟（00-59）
- `%n`：换行符
- `%p`：AM/PM
- `%P`：am/pm
- `%r`：本地时间（12小时制）
- `%R`：时间（24小时制）
- `%s`：Unix时间戳
- `%S`：秒（00-60）
- `%t`：制表符
- `%T`：时间（24小时制）
- `%u`：星期几（1-7）
- `%U`：一年中的第几周（以周日为每周的第一天）
- `%V`：一年中的第几周（以ISO 8601标准为准）
- `%w`：星期几（0-6）
- `%W`：一年中的第几周（以周一为每周的第一天）
- `%x`：本地日期
- `%X`：本地时间
- `%y`：年份的后两位
- `%Y`：年份的完整形式
- `%z`：时区偏移量，如+0800
- `%Z`：时区名称

例如，要显示当前时间，可以使用以下命令：

```
gdate '+%Y-%m-%d %H:%M:%S'
```

要设置系统时间，可以使用以下命令：

```
sudo gdate -s '2021-06-25 12:34:56'
```
