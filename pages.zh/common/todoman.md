# todoman 
## chatgpt 
todoman是一个命令行工具，提供了一个简单的任务管理系统，允许用户追踪待完成的任务。

该命令的具体语法如下：

```
todoman [OPTIONS] COMMAND [ARGS]...
```

其中，OPTIONS是可选参数，可以使用--help选项来查看具体的选项。COMMAND是要执行的命令，以下是可用的命令：

1. add：添加新的任务。

2. list：列出当前待办任务。

3. start：开始处理任务。

4. complete：完成任务。

5. cancel：取消任务。

6. edit：编辑任务。

7. delete：删除任务。

8. search：搜索任务。

除了以上命令外，还可以使用以下通用选项：

1. --version：显示程序版本号。

2. --help：显示帮助文档。

以上就是todoman命令的详细解释。
